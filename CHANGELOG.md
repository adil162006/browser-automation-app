# Changelog

All notable changes to this project are documented here. The repository has no
Git tags as of July 20, 2026, so the first section describes the current
`0.0.1` development snapshot rather than a published release.

The format follows Keep a Changelog conventions.

## [Unreleased]

### Known

- Automated tests and CI/CD are not implemented.
- ESLint currently reports two `react-hooks/set-state-in-effect` errors.
- No open-source license has been selected.

## [0.0.1] - 2026-07-20

### Added

- Next.js 16 App Router application shell and Docker standalone deployment.
- Clerk sign-in, sign-up, organization selection, organization switching, and
  organization billing.
- Neon Postgres and Drizzle workflow persistence.
- Liveblocks collaborative React Flow canvas, cursors, presence, and user
  resolution.
- Registry-driven workflow nodes for Start, Open URL, Act, Extract, Observe,
  Agent, and Send Email.
- Upstream output interpolation and graph validation.
- Trigger.dev workflow execution, live step metadata, cancellation, logs,
  output inspection, and timing.
- Browserbase/Stagehand browser execution and Pro-only HLS session replay.
- Resend email execution node.
- Theme support and workflow design reference images.
